# Documentación - AGENTES-Y-SKILLS

Este directorio centraliza toda la documentación de referencia para humanos y los protocolos operativos para los Agentes de IA.

## Contenido Esperado

*   **Manuales Operativos:** Definiciones de comportamiento y restricciones para la IA (ej. `manual-operativo.md`).
*   **Guías de Usuario:** Tutoriales paso a paso y flujos de trabajo (ej. `guia-orden-instrucciones.tex` y su PDF compilado).
*   **Referencias:** Documentación técnica adicional.

> **Nota:** Según el `Agent.md`, los manuales operativos deben residir o enlazarse aquí para mantener la raíz del proyecto limpia.
